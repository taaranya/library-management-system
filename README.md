# Library Management System

This is a menu-driven Library Management System developed using C programming.
The system uses file handling to store and manage book records permanently.

## Objective
To design and implement a Library Management System that helps in managing
book records such as adding books, issuing books, returning books, and deleting records.

## Features
- Add new books
- Display all books
- Issue books
- Return books
- Delete book records
- File-based data storage

## Technologies Used
- C Programming
- File Handling
